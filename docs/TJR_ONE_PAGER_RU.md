# TJR: Telegram Job Radar

## Что это
`TJR` — desktop-приложение для macOS и Windows, которое помогает находить релевантные вакансии в Telegram-каналах и чатах под вашей учетной записью.

Приложение сканирует выбранные источники, оценивает каждое сообщение по заданным критериям и показывает только подходящие совпадения в удобном отчете.

## Какую проблему решает
- Ручной просмотр десятков каналов с вакансиями занимает много времени.
- В потоке много “шума”: рекомендации кандидатов, обучение/курсы, нерелевантные роли.
- Важные вакансии легко пропустить из-за скорости публикаций.

`TJR` автоматизирует этот процесс и снижает операционные затраты на мониторинг вакансий.

## Для кого продукт
- Топ-менеджеры и руководители, ищущие роли уровня `CEO/CPO/COO/Директор`.
- Карьерные консультанты и executive search-специалисты.
- Пользователи, которым нужен персональный “радар” по рынку труда в Telegram.

## Ключевая ценность
- Экономия времени: автоматический скан вместо ручного чтения ленты.
- Точность: фильтрация по названию, профилю, отрасли + исключающие фразы.
- Контроль: прозрачный прогресс сканирования и быстрый доступ к последнему отчету.
- Непрерывность: настройки сохраняются локально и переиспользуются между версиями.

## Что умеет сейчас (v2)
- Авторизация и работа с Telegram через пользовательскую учетку (API ID/API Hash/телефон).
- Выбор чатов и каналов для мониторинга.
- Фильтрация сообщений по:
  - совпадению по названию роли;
  - совпадению по профилю;
  - совпадению по отрасли;
  - исключающим фразам.
- Поддержка словоформ (склонений) для русского языка.
- Live-интерфейс сканирования:
  - счетчик найденных вакансий;
  - статус текущего канала и прогресса;
  - `MATCH FEED` с пополнением в реальном времени.
- Кликабельные ссылки на оригинальные сообщения Telegram.
- Отчет с сортировкой по дате, бан-листом сообщений и экспортом в `.xlsx`.
- Отдельный быстрый блок настроек на главном экране:
  - каналы/чаты;
  - совпадение по названию;
  - глубина поиска (в днях);
  - исключения.

## Безопасность и данные
- Конфигурация хранится локально:
  - macOS: `~/Library/Application Support/TJR/config.json`
  - Windows: `%APPDATA%\\TJR\\config.json`
- Логи также хранятся локально и очищаются при новом запуске сканирования.
- В публичный репозиторий и релизы публикуется только `empty-config.json` (пустой шаблон), без пользовательских данных.

## Совместимость версий
- Версия `v2` совместима с настройками `v1`.
- Пользователи `v1` продолжают работать с тем же файлом конфига и не вводят данные заново.
