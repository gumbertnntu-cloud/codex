# TJR для macOS: Быстрый старт

## 1. Скачивание
- Откройте релиз `v2`: `https://github.com/gumbertnntu-cloud/codex/releases/tag/v2`
- Скачайте архив `TJR-macOS.zip`.
- Распакуйте архив и переместите `TJR.app` в `Applications`.

## 2. Первый запуск
- Запустите `TJR.app`.
- Если macOS покажет предупреждение о неизвестном разработчике:
  - `Control + click` по `TJR.app` -> `Open` -> `Open`.

## 3. Пустой файл настроек
- Шаблон пустого конфига: `empty-config.json` (в этом же релизе).
- Рабочий путь конфига приложения:
  - `~/Library/Application Support/TJR/config.json`

Если файл отсутствует, приложение создаст его автоматически при первом запуске.

## 4. Где взять Telegram API ID и API Hash
- Перейдите: `https://my.telegram.org/auth?to=apps`
- Создайте приложение и скопируйте:
  - `api_id`
  - `api_hash`

## 5. Что заполнить в приложении
- На главном экране (слева):
  - `Каналы/чаты`
  - `Совпадение по названию`
  - `Глубина поиска`
  - `Исключения`
- В `Настройки`:
  - `Telegram API ID`
  - `Telegram API Hash`
  - `Телефон Telegram`
  - Профиль и отрасль (если нужно точнее фильтровать)

## 6. Совместимость с v1
- Версия `v2` использует тот же путь и формат конфига:
  - `~/Library/Application Support/TJR/config.json`
- Если в `v1` данные уже были введены, в `v2` повторный ввод не требуется.
